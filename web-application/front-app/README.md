## Setup

Install [Nvm](https://github.com/nvm-sh/nvm) and Node 
```
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
$ nvm install 16.14.2
$ nvm unalias default
$ nvm alias default 16.14.2
$ node -v
v16.14.2
```

Install packages
```
$ npm install
```

Run
```
$ npm run start
```

Build
```
$ npm run build
```

Test
```
$ npm run test
```
